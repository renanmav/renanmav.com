This article is a step-by-step guide to integrating Passkeys and smart contract wallets into a React Native project. We'll use `react-native-passkey` for Passkey authentication and `viem` for smart contract interactions.

## Prerequisites

- Basic knowledge of React Native
- Node.js and npm installed
- Expo CLI installed (if using Expo)
